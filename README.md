# Non-Powered Dams Hydropower Analysis Project

## Description
Assessing the potential of non-powered dams in the United States for hydropower 

## Data Collection
- Will update 

## File Organization
- `raw_data/` - Original logger files
- `processed_data/` - Cleaned CSV files
- `analysis/` - R and Python scripts
- `metadata/` - Data documentation
- `outputs/` - Figures and reports
- `archive/` - Legacy Data

## Requirements
- Python 3.8+
- Packages: pandas, matplotlib, numpy
